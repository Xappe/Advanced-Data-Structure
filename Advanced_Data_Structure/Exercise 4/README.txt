Implemente (na Linguagem C) os seguintes exemplos de Função de Hashing: Método da Divisão,
Método da Multiplicação e Método da Dobra (a partir de chaves com 8 dígitos, obter endereços de
2 dígitos). Cada função deve receber por parâmetro a chave e o tamanho da tabela.

Implement (in C Language) the following examples of Hashing Function: Division Method,
Multiplication Method and Fold Method (from 8-digit keys, get addresses from
2 digits). Each function must receive the key and size of the table as a parameter.
