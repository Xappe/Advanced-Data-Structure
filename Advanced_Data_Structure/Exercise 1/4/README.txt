Considere um grafo simples não-orientado G com n vértices e m arestas armazenado
utilizando sua Matriz de Incidência M. Crie uma função (na Linguagem C) que receba por
parâmetro a matriz M, bem como n e m, e imprima os graus de cada vértice de G.

Consider a simple undirected graph G with n vertices and m edges stored
using its Incidence Matrix M. Create a function (in C Language) that receives by
parameter the matrix M, as well as n and m, and print the degrees of each vertex of G.
