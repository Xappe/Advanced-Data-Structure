Faça um algoritmo recursivo (sem utilizar qualquer estrutura de repetição) que leia a altura h (h ≥ 0)
de uma árvore AVL e imprima o número mínimo de nós para construir a referida árvore.

Make a recursive algorithm (without using any repetition structure) that reads the height h (h ≥ 0)
of an AVL tree and print the minimum number of nodes to build said tree.
